# Lista_Canales_LG_Samsung
¡Bienvenido al **Channel List Generator**! Esta es una herramienta web de un solo archivo (SPA) desarrollada para administrar, editar masivamente y exportar mapas de canales para sistemas de televisión interactiva hotelera, específicamente: - **Samsung LYNK Cloud** (Archivos `.json`) - **LG Pro:Centric Direct** (Archivos `.xml`)
